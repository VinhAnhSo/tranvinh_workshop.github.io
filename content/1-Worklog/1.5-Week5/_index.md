- Understand AWS Lambda functions and event triggers.

- Practice basic DynamoDB operations for serverless applications.
- Build a hands-on REST API backend using serverless services.

### Week 5 Achievements:
- Gained a clear understanding of AWS Lambda fundamentals, including how functions are created, triggered, and executed in a serverless environment.
- Learned how API Gateway works as an entry point for serverless applications and how it integrates with Lambda to expose RESTful API endpoints.
- Practiced basic DynamoDB operations such as creating tables, inserting data, retrieving records, updating items, and deleting data as part of backend processing.
- Improved understanding of DynamoDB table design, especially the role of partition keys and how data is structured for efficient access.
- Explored the interaction flow between Lambda, API Gateway, and DynamoDB to better understand how AWS serverless services work together in a complete backend architecture.
- Successfully built and tested a basic REST API backend using AWS serverless services, including request handling, business logic execution, and database interaction.
- Strengthened practical skills in designing lightweight backend systems without managing traditional servers.
- Increased overall understanding of serverless architecture concepts, deployment flow, and the benefits of scalability, flexibility, and reduced infrastructure management in AWS.
