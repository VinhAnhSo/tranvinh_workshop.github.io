### Week 10 Achievements:
- Developed a clearer understanding of AWS DevOps services and how they support continuous integration and continuous deployment in modern cloud environments.
- Learned the roles of CodeCommit, CodeBuild, and CodeDeploy in the software delivery lifecycle, from source code management to build automation and application deployment.
- Explored AWS CodePipeline and understood how different stages can be connected to automate the flow from code changes to deployment outcomes.
- Improved practical understanding of CI/CD concepts by studying how automated pipelines reduce manual effort, improve consistency, and accelerate release processes.
- Gained knowledge of Infrastructure as Code principles and learned how AWS CloudFormation can be used to provision and manage infrastructure resources in a repeatable and declarative way.
- Practiced working with CloudFormation templates and stack-based deployment approaches, which helped reinforce the relationship between application deployment and infrastructure automation.
- Explored how AWS DevOps tools integrate together to form a complete pipeline that covers version control, build, deployment, and infrastructure provisioning.
- Successfully built a basic automated deployment pipeline and gained hands-on experience with orchestrating the end-to-end workflow of code delivery on AWS.
- Strengthened overall understanding of DevOps practices in cloud environments, especially in the areas of automation, deployment consistency, infrastructure management, and operational efficiency.
