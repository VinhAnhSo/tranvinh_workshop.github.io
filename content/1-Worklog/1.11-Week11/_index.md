### Week 11 Objectives:
- Understand AWS WAF and Shield for application protection.
- Learn how to manage sensitive credentials using AWS Secrets Manager.
- Explore AWS Certificate Manager for SSL/TLS certificate management.
- Build a hands-on secure production environment using AWS security services.

### Week 11 Achievements:
- Developed a stronger understanding of advanced AWS security services and how they contribute to protecting production environments from both external threats and internal misconfigurations.
- Learned the core functions of AWS WAF and understood how web access rules can be used to filter malicious traffic, restrict unwanted requests, and improve application-level security.
- Explored AWS Shield and gained knowledge of DDoS protection concepts, including how AWS helps maintain service availability and resilience against large-scale attack scenarios.
- Improved understanding of the relationship between AWS WAF and Shield, especially in the context of layered security for internet-facing applications.
- Studied AWS Secrets Manager and learned how sensitive information such as API keys, passwords, and connection strings can be stored and managed securely instead of being hardcoded in applications.
- Practiced secure secret handling concepts and strengthened awareness of best practices such as controlled access, centralized secret storage, and secret rotation.
- Explored AWS Certificate Manager and learned how SSL/TLS certificates are provisioned, managed, and applied to support encrypted communication in cloud environments.
- Improved practical understanding of how certificate management supports secure connections, trust establishment, and safer deployment of production-grade services.
- Successfully combined multiple AWS security services in a basic secure production environment, gaining hands-on experience in applying protection, credential security, and certificate-based encryption together.
- Strengthened overall awareness of cloud security best practices, especially in the areas of web protection, traffic resilience, secure secret management, and encrypted communication for production systems.
